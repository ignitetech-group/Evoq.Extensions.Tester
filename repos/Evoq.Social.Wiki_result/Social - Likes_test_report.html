Test 1: Like an Article PASS

Objective: Verify that a user can like a wiki article.

Steps Taken:
  1. Navigated to Wiki module (Community > Wiki)
  2. Clicked on "Published Article Test - Jan 2026" article
  3. Located the "Like this" button (thumbs up icon)

Actual Result:

  • "You like this" text appeared below article content

Observations

  • Scoring Points: Liking an article awards the user 5 reputation/experience points. This was observed when user points increased from 79 to 84 after liking.
  • Like Display Format: The like count is displayed as "You like this" when the current user has liked the article. For multiple users, it would likely show "You and X others like this".
  • Code Review Finding: The code review confirmed that like/unlike functionality is handled by the DNN Social Library (socialController.like() and socialController.unlike() methods in dnn.wiki.Article.js). The Wiki module integrates with this library for social features.
  • Note on Suggested Scenarios: The following suggested scenarios were not tested as they require multiple user accounts or admin configuration which is beyond standard testing scope for Medium priority:
    • Test multiple users liking (requires multiple user sessions)
    • Test like permissions (requires permission configuration)
    • Test like scoring points (partially tested - points increase was verified)
