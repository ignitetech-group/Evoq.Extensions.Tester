Observations

  • Code Analysis - IPortable Not Implemented: The Wiki module's BusinessController class (BusinessController.cs) only implements ModuleSearchBase and IUpgradeable interfaces. It does NOT implement the IPortable interface which is required for DNN's standard Import/Export Content functionality.
  • Module Definition Verification: The Wiki.dnn manifest file shows supportedFeatures only includes "Upgradable" and "Searchable" - NOT "Portable". This confirms at the package level that Import/Export is not supported.
    <supportedFeatures> <supportedFeature type="Upgradable" /> <supportedFeature type="Searchable" /> </supportedFeatures>
  • ContentExchangeController Exists But Different Purpose: The code does include a ContentExchangeController.cs file, but this is for "Content Exchange" functionality (moving content between modules like Discussions to Wiki), NOT for Import/Export backup/migration operations.
  • Other Social Modules DO Support Import/Export: Some other Evoq Social modules (like Discussions, Ideas, MyStatus) DO implement IPortable and support Import/Export. The Wiki module appears to be an exception where this feature was not implemented.
  • Feature Documentation Mismatch: The test specification suggests this feature should exist at "Module Actions > Import/Export Content", but the actual implementation does not include this functionality. This may indicate outdated documentation or a feature that was planned but not implemented.

Conclusion

The Import/Export Content feature for the Wiki module is NOT IMPLEMENTED. This is a FAIL result based on SMOKE testing requirements to verify "the feature loads and is accessible."

The feature cannot be tested because:

  • The IPortable interface is not implemented in the Wiki module's code
  • The module definition does not register "Portable" as a supported feature
  • No Import/Export options appear in the Module Actions menu in the UI

Recommendation: If Import/Export functionality is required for Wiki articles, a feature request should be submitted to implement the IPortable interface in the Wiki module.
