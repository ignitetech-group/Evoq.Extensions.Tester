Recommendations

Critical Fix Required

The Utilities.FilterMaliciousTags() method must be updated to sanitize ALL event handlers, including:

  • onerror
  • onload (on img, body, iframe, etc.)
  • onmouseover, onmouseout
  • onfocus, onblur
  • oninput, onchange
  • All other JavaScript event handlers

Suggested approach: Use a whitelist approach where only explicitly allowed attributes are preserved, rather than a blacklist approach that tries to remove known dangerous attributes.

// Recommended: Strip ALL on* attributes from all elements foreach (var element in doc.DocumentNode.Descendants()) { var attributesToRemove = element.Attributes .Where(a => a.Name.StartsWith("on", StringComparison.OrdinalIgnoreCase)) .ToList(); foreach (var attr in attributesToRemove) { attr.Remove(); } }
