File Browser - Test Report

Extension: Evoq.PersonaBar.UI (PersonaBar Module)

Feature Name: File Browser

Description: Allows users to browse, search, and filter files including images, documents, and other file types within the portal.

Feature Priority: High

UI Location: Admin > PersonaBar > Content > Assets/Files

Test Environment: http://localhost:8081

Test User: host (SuperUser)

Test Scenarios

1. Search Files by Keyword PASS

Steps:
  1. Navigate to Content > Assets in PersonaBar
  2. Click on "My Website" to access the site's assets
  3. Enter "test" in the Search Files box
  4. Press Enter to search

Search results showing files containing "test" keyword

Observations:

Search successfully found multiple files matching the keyword "test" including test_document.docx, test_document.pdf, test_image.bmp, test_image.gif, test_image.jpg, test_image.png, test_spreadsheet.xlsx, and test_workflow_file.txt. The breadcrumb correctly shows "Search 'test'" to indicate an active search filter.

2. Filter Files by Folder PASS

Steps:
  1. Click on the folder dropdown (Home)
  2. View the folder tree structure
  3. Select "Images" folder from the tree
  4. Verify the contents are filtered to show only Images folder content

Files filtered to show Images folder contents

Observations:

The folder navigation dropdown displays a hierarchical tree of available folders. Selecting the Images folder successfully filters the view to show only files and subfolders within the Images directory, including the Publisher subfolder and various image files (cavalier-tv.png, CTA-1.jpg through CTA-8.jpg, etc.).

3. View Recent Files (Sort by Last Modified) PASS

Steps:
  1. Locate the Sort dropdown (initially set to "Alphabetical")
  2. Click on the Sort dropdown
  3. Select "Last Modified" option
  4. Verify files are sorted by most recently modified first

Files sorted by Last Modified date (most recent first)

Observations:

The sort dropdown provides three options: Last Modified, Date Uploaded, and Alphabetical. When "Last Modified" is selected, files are reordered with the most recently modified files appearing first. The sort order correctly updates the file grid display.

4. Navigate Folder Hierarchy PASS

Steps:
  1. From the Images folder, click on the "Publisher" subfolder
  2. Verify navigation into the subfolder
  3. Verify breadcrumb shows correct path (Home > Images > Publisher)
  4. Click on "Images" in the breadcrumb to navigate back
  5. Verify successful navigation back to parent folder

Navigated into Publisher subfolder showing breadcrumb trail

Observations:

Folder hierarchy navigation works correctly. Double-clicking a folder navigates into it, and the breadcrumb trail updates to show the full path (Home > Images > Publisher). Clicking on any breadcrumb item navigates directly to that folder level. The Publisher folder contains various thumbnail images and test files.

5. File Preview Functionality PASS

Steps:
  1. Hover over a file card (cavalier-tv.png)
  2. Verify action icons appear on hover
  3. Observe available actions (Edit, Move, Copy, Download, Delete)
  4. Verify thumbnail preview is visible

File card hover state showing action icons and thumbnail preview

Observations:

Hovering over a file card reveals action icons on the right side of the card: Edit (pencil), Move (arrow box), Copy (overlapping squares), Download (down arrow), and Delete (trash can). File thumbnails are displayed for image files, providing a visual preview of the content. Single-clicking a file triggers a download, while the edit icon provides access to file details.

6. Special Characters in Search PASS

Steps:
  1. Clear any existing search text
  2. Enter "CTA-1" (containing hyphen special character) in search box
  3. Press Enter to execute search
  4. Verify search correctly handles the hyphen character

Search results for "CTA-1" (with hyphen special character)

Observations:

Search functionality correctly handles special characters like hyphens. Searching for "CTA-1" successfully found and displayed CTA-1.jpg as the matching result. The search term is preserved in the search box and reflected in the breadcrumb trail.
