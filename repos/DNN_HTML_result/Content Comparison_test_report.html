Observations

  • UI Location Note: The Content Comparison feature is accessed through the Page History icon in the edit bar, not through Module Actions as originally documented. This is a page-level feature that compares entire page versions, not individual module content versions.
  • Comparison Mechanism: The comparison uses the HtmlDiff algorithm (found in HtmlDiff.dll) which generates visual diffs with <ins> and <del> HTML tags and CSS classes (diffins, diffdel).
  • Version Limit: Only 2 versions can be selected for comparison at a time. When 2 versions are selected, other checkboxes become disabled.
  • View Format: Comparison displays versions in a stacked format (newer version on top, older on bottom) rather than true side-by-side.
  • Export/Navigation Features: The UI does not show explicit "Export comparison report" or "Navigate between differences" options as suggested in test scenarios. These may not be implemented or may require different access paths.
