Test Report: Engagement and Influence Metrics

Extension: Evoq.PersonaBar.Users (PersonaBar Module)

Feature Priority: Medium

Test Date: January 6, 2026

UI Location: Admin > Users > Select User > Activity Tab > Engagement/Influence Section

Feature Description

Calculate and display user engagement and influence percentages. This feature is available only for the DNNSOCIAL SKU (Evoq Engage/Social).

Engagement Metric Calculated from XML time series data, compared against optimal engagement score, displayed as 0-100%
Influence Metric Retrieved from Analytics_UserInfluenceRollup table, displayed as a percentage value
SKU Requirement Metrics only display when SKU is DNNSOCIAL (Evoq Engage/Social editions)
Dependencies Analytics_Users table, Analytics_UserInfluenceRollup table

-- From Install.SqlDataProvider
--Influence (available in Social only)
IF EXISTS (SELECT * FROM sys.objects WHERE object_id = OBJECT_ID(N'{databaseOwner}[{objectQualifier}Analytics_UserInfluenceRollup]'))
    SELECT InfluenceYearly Influence
    FROM {databaseOwner}{objectQualifier}Analytics_UserInfluenceRollup
    WHERE UserId = @UserId
ELSE
    SELECT -1 Influence
