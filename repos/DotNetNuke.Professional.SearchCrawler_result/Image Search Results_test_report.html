Technical Notes

Code Implementation Details

Based on code review of the relevant files:

  • ImageResultController.cs - Simple controller extending FileResultController that provides a localized search type name for images via the LocalizedSearchTypeName property
  • FileCrawlerController.cs - Contains main indexing logic:
    • Uses _imageSearchTypeId and _documentSearchTypeId to categorize files
    • GetSearchTypeId() method (lines 449-453) determines if a file is an image based on Globals.glbImageFileTypes
    • ImageExtensions property returns image types from DNN global configuration
    • PopulateSearchDocument() adds metadata including fileName, extension, contentType, folderName
    • CleanExistingIndexedImageAsDocument() handles cleanup when an image was previously indexed as document
