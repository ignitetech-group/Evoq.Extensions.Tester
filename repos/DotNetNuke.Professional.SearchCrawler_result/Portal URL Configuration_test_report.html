Observations

  • UI Behavior: Clicking edit or delete icons opens the edit form for the URL. The delete action may require confirmation or may be handled within the edit view.
  • Code Features: The SearchSpiderHelper.cs file contains methods for:
    • GetPortalUrlCrawlAliases() - Retrieves portal-specific URL configurations
    • InitializePortalUrlCrawlAliases() - Auto-initializes URLs using primary portal alias with SSL detection
    • GetPortalId() - Resolves portal ID from URL by matching against portal aliases
  • SSL Detection: The code checks GetPortalSetting(portalId, "SSLEnabled") to determine the appropriate protocol (http/https).
  • Windows Authentication: The UI supports optional Windows Authentication credentials for URL crawling (Domain, User Account, Password).
  • Enable Spidering: Each URL can have spidering individually enabled or disabled.
  • Duplicates Section: The Crawling tab also includes regex patterns for handling duplicate content detection.
