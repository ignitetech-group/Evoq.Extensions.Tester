Code-Level Portal Isolation Analysis

Review of the source code reveals that portal-specific analytics IS properly implemented at the code level:

File: Services/CommunityAnalyticsController.cs

The API controller implements portal isolation through the EffectivePortalId property:

protected int EffectivePortalId { get { if (_effectivePortalId == Null.NullInteger) { _effectivePortalId = ActiveModule != null ? ActiveModule.OwnerPortalID : PortalController.GetEffectivePortalId(PortalSettings.PortalId); } return _effectivePortalId; } }
