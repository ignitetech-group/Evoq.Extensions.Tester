  • Menu Structure: Community Analytics is available under Dashboard > Community Analytics in the PersonaBar menu structure.

Recommendations

  1. Configure Analytics Warehouse: Ensure the Analytics Warehouse dependency is properly set up and connected before testing this feature.
  2. Fix JavaScript Error: Investigate and fix the utility.serializeCustomDate is not a function error in the Community Analytics dashboard code.
  3. Improve Error Handling: The API should return meaningful error messages instead of empty 500 responses to help with debugging.
  4. Documentation: Update feature documentation to clarify the prerequisite setup steps for Analytics Warehouse.
