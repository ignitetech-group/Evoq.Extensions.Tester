Test 6: Analytics Cookies

PASS

Objective: Verify analytics cookies are set for tracking visitors.

Steps Taken:
  1. Evaluated document.cookie in browser console
  2. Checked for GA4 and legacy DNN analytics cookies

Cookie Values Found:

_ga=GA1.1.1864371918.1767716573 _gid=GA1.1.2089203129.1767716573 _ga_ABC123DEF4=GS2.1.s1767716578$o1$g0$t1767716586$j52$l0$h0

Analysis:

  • _ga - Google Analytics client ID cookie (present)
  • _gid - Google Analytics session cookie (present)
  • _ga_ABC123DEF4 - GA4 measurement-specific cookie (present)
  • Analytics_VisitorId - Legacy DNN visitor cookie (NOT present - legacy tracking disabled)
  • Analytics - Legacy DNN session cookie (NOT present - legacy tracking disabled)

Observations

  • Legacy Analytics Disabled: The internal DNN page view tracking (via LogPageView API) is not active. The code shows this is controlled by the EnableLegacyAnalyticsCookies web.config setting and AreLegacyAnalyticsCookiesEnabled() function. The dnn.analytics.Injected.js script is not loaded on pages.
  • GA4 Integration Working: Google Analytics 4 is successfully tracking page views via the connector. The measurement ID (G-ABC123DEF4) is configured and network requests show successful tracking.
  • OAuth Authorization Required: For full Page Analytics dashboard functionality, Google OAuth authorization is required but not completed in the test environment.
  • Code Infrastructure Present: The server-side tracking infrastructure (AnalyticsServiceController.cs, AnalyticsController.cs, AnalyticsProvider.cs) is present and tracking variables are being set, but the client-side JavaScript to send data to the LogPageView API is not loaded.
  • Session Timeout Configuration: Code shows max time on page defaults to 300 seconds (5 minutes) via Analytics_MaxSecondsTimeOnPageKey portal setting.
  • Excluded Tabs Feature: Code supports excluding tabs from tracking via ContentItemUrlProvider_ExcludeTabIds portal setting, but this could not be tested via UI as no configuration interface was found.

Recommendations

  1. Fix Language Tracking Bug: The evoq_PageLanguage variable should update when navigating to different language versions of pages.
  2. Complete Google OAuth Setup: For full analytics dashboard functionality, complete the Google OAuth authorization process.
  3. Document Legacy Analytics: Clarify whether legacy internal analytics (LogPageView API) is intentionally disabled or should be enabled via EnableLegacyAnalyticsCookies setting.
  4. Add Excluded Tabs UI: Consider adding a UI interface to configure excluded tabs for analytics tracking.
