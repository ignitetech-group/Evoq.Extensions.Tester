Code Analysis

The relevant API endpoint was reviewed at:

src/Modules/Evoq.PersonaBar.UrlManagement/Services/UrlManagementServiceController.cs

The GetConfiguration() method (lines 486-495) returns:

  • Success: Boolean indicating successful retrieval
  • SingleLocale: Boolean flag indicating if portal has single locale or content localization enabled
  • SortingFields: Enum values (None, Url, Locale, Status) for sorting URL lists

The UI displays additional configuration from the DNN FriendlyUrlSettings system, providing a comprehensive view of all URL management options.
