Feature Overview

The User Identity Mapping feature provides a database-level mapping between Active Directory User Principal Names (UPNs) and DNN usernames. This enables flexible authentication where AD identities can be associated with DNN user accounts.

Key Components

  • AD_UserMapping Table: Stores the mapping between IdentityName (UPN) and UserId
  • AD_GetUserMapping: Stored procedure to retrieve DNN username by AD identity
  • AD_UpdateUserMapping: Stored procedure to create/update mappings
  • CASCADE Delete: Automatic cleanup when users are deleted
